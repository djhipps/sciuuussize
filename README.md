# SciuuuSize Plugin

Custom size guide plugin for SciuuuS Kids WooCommerce store.

## Description

Provides an interactive size guide modal for children's barefoot shoes, helping customers find the perfect fit.

## Features

- Interactive size guide modal
- Age-to-size recommendations
- WooCommerce integration
- Mobile-responsive design

## Installation

1. Upload to `wp-content/plugins/sciuuussize/`
2. Activate through WordPress admin
3. Configure in WooCommerce → Settings → SciuuuSize

## Version

Current version: 1.0.0
